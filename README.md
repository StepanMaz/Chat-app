# Description
Simple chat app written with **nest.js**, **react**, **typeorm**, **docker-compose**, **postgresql**


# Instalation
```
$ npm install
```

# Services
```
# start services
$ npm run services:up

# stop services
$ npm run services:down

# stop services and clear data
$ npm run services:down-clear
```

# Running this app
```
# single command to start everithing right after git clone
# installs everething
# generates .env
# starts services
# starts app in production mode
$ npm start 

# development
$ npm run dev

# porduction mode
$ npm run start:prod
```

# Test
```
$ npm run test
```
